開始忙碌的新一年

January 31, 2017

在這良辰雞年，為了繼續為我們的藝術家們紀錄炒雞棒的紋身作品，為了在研究的道路上大步向前， 我們將要為大家介紹一大堆新的功能。在技​​術方面，網站添加了一些吸引人的新功能。

首先，公告頁面可以有帶有自己鏈接的單獨頁面了:

  • gogo.tattoo/post/2016-11-27-vision
  • gogo.tattoo/post/2016-12-17-annual-summary

同時，紋身藝術家們的個人頁面也變得越來越好看了。我們覺得雞年是一個非常好的時間來讓我們在原有的基礎上優化我們的社區。 我們現在的目標是在一些鏈接裡介紹我們的最新功能。

頂端的標籤

紋身作品現在都帶有標籤了，所以來訪者可以按照自己的標籤喜好來找到並關注相應紋身藝術家， 也可以讓紋身藝術家更加容易的在自己的工作歷史裡找到有關標籤的相應紋身作品。 標籤會在藝術家的個人頁面頂端和紋身作品頁面的底部顯示。

訪問gogo.tattoo的用戶們可以使用這個簡易的功能輕鬆找到他們正在尋覓的設計樣式， 和擅長此類樣式的藝術家。在透明化紋身樣式偏好的趨勢裡我們因此而前進了巨大一步， 使gogo.tattoo向我們設想的方向發展。

  • gogo.tattoo/gogo/tags/dragon
  • gogo.tattoo/aid/tags/color
  • gogo.tattoo/gogo/tags/watercolor
  • //gogo/tattoo/anchored_infinity_moon

全新的紋身手稿頁面

這是另一個幫助我們的藝術家與在此頁面上欣賞他們作品的來訪者們相連接的方法。 這是一個可以讓藝術家的創作更有辨識度並讓藝術家們展示他們下一個想要製作的紋身作品的手稿的地方。

  • gogo.tattoo/gogo/design

Github 代碼庫是可用的

對於想要加入gogo.tattoo的程序員們和尋求合作的伙伴們來說，我們的Github代碼庫裡充滿了我們網站建設中的更新和進程。 網站demo/alpha版本 裡的藝術家頁面現在是開源的。

Gogo-upload

Gogo-upload是用Golang寫的程序。它的目的是為照片添加帶有標記的水印，調整其大小並上傳到IPFS。 此程序已經完成了一半了，我們邀請您來促進次程序的發展。
