让我们一步一脚印

在过去几个月里，在区块链的世界里发生了许多令人兴奋的新闻，这些迷人又可怕的新闻吸引了我们的注意。智能合约正在蓬勃发展，这一强大的新机制正获得更广泛的关注，但同时此机制的性能问题造成了一些诸如以太坊区块链的最新分支之类的让人紧张不安问题。我们所有的人都在密切注视。

针对上述情况，gogo.tattoo正在密切监测在对等网络生态系统中的大环境，并继续前进与发展。我们觉得更重要的是要用长远的眼光看前面的路，而不是不顾一切的轻率开始行动。我们正在进入一个还没有被技术人员知根知底的全新领域 — 让想法变成现实的区块链。

我们很高兴我们正在建设发展中，其中的项目包括建设我们的纹身艺术家简介页，在那里，你将能够将你的工作作为去中心化的一部分展示。这是世界各地的艺术家和纹身爱好者的网络。我们非常期待能为您带来这一切。我们一如既往希望有志同道合的人可以一起合作。

明年的我们会更酷

如果你在 gogo.tattoo 今年推出以来一直在关注我们，你可能会有一些关于如果将我们使命变成焦点的想法意见。在一方面，建立开发的这个将纹身经验与充满可能性的区块链相连接的平台是首要任务；另一方面，我们正在努力的在线下纹身世界中让大家了解到我们的存在，我们从与当地纹身工作室和纹身艺术家交流开始，让他们意识到我们正在朝着什么样方向发展.。

最近的最新消息总是令人兴奋的，无论你最关心的是纹身还是网页内部技术（两者都会被永久保存）。我们在一个令人兴奋的城市里的一个秘密地点内工作。这个城市里聚集着区块链世界里的主要工作者，大家都在此技术领域里投资了自己的时间和精力，专注于这项可以让互联网世界改革的项目。再加上在区块链圈子里我们十分有前途的关注度，还有我们从那些从事纹身事业的人那里得到的各种欢迎，这些都推动着我们前进，以一种无法预期的超快速度逐步将我们的愿景变成现实。这个月我们现场发布了一则纹身项目的工作证明，对于我们来说这是一个重要的里程碑。这可令人非常兴奋，这个经验继续揭示了 gogo.tattoo 可以使纹身艺术家受益，也能使选择了他们的客户们受益，并让允许他们进行纹身的工作室也受益。我们一直都知道我们能为人们做些什么，但这意味着我们也需要接收更多人的反馈。

我们很高兴能与众多在幕后对我们的项目充满热情的人们一起步入新的一年。这是我们帮助着推动的一个短暂的，快速的，野蛮生长的旅程。接下去将会发生的事将会是惊奇的，是惊喜的，也是带有启发性的。我们期待着在新的一年里我们继续在这个创意世界的酷炫一角继续一起改变世界。

开始忙碌的新一年

在这良辰鸡年，为了继续为我们的艺术家们纪录炒鸡棒的纹身作品，为了在研究的道路上大步向前， 我们将要为大家介绍一大堆新的功能。在技术方面，网站添加了一些吸引人的新功能。

首先，公告页面可以有带有自己链接的单独页面了:

  • gogo.tattoo/post/2016-11-27-vision
  • gogo.tattoo/post/2016-12-17-annual-summary

同时，纹身艺术家们的个人页面也变得越来越好看了。我们觉得鸡年是一个非常好的时间来让我们在原有的基础上优化我们的社区。 我们现在的目标是在一些链接里介绍我们的最新功能。

顶端的标签

纹身作品现在都带有标签了，所以来访者可以按照自己的标签喜好来找到并关注相应纹身艺术家， 也可以让纹身艺术家更加容易的在自己的工作历史里找到有关标签的相应纹身作品。 标签会在艺术家的个人页面顶端和纹身作品页面的底部显示。

访问gogo.tattoo的用户们可以使用这个简易的功能轻松找到他们正在寻觅的设计样式， 和擅长此类样式的艺术家。在透明化纹身样式偏好的趋势里我们因此而前进了巨大一步， 使gogo.tattoo向我们设想的方向发展。

  • gogo.tattoo/gogo/tags/dragon
  • gogo.tattoo/aid/tags/color
  • gogo.tattoo/gogo/tags/watercolor
  • //gogo/tattoo/anchored_infinity_moon

全新的纹身手稿页面

这是另一个帮助我们的艺术家与在此页面上欣赏他们作品的来访者们相连接的方法。 这是一个可以让艺术家的创作更有辨识度并让艺术家们展示他们下一个想要制作的纹身作品的手稿的地方。

  • gogo.tattoo/gogo/design

Github 代码库是可用的

对于想要加入gogo.tattoo的程序员们和寻求合作的伙伴们来说，我们的Github代码库里充满了我们网站建设中的更新和进程。 网站demo/alpha版本 里的艺术家页面现在是开源的。

Gogo-upload

Gogo-upload是用Golang写的程序。它的目的是为照片添加带有标记的水印，调整其大小并上传到IPFS。 此程序已经完成了一半了，我们邀请您来促进次程序的发展。
